Stripe Keys in GitHub

Public repository incident

Problem

Stripe secret key committed to public GitHub repo. Anyone could issue refunds and access customer-level billing data.

Solution

  • Moved secrets to environment variables

Result: No fraudulent activity detected. Team avoided a potentially catastrophic breach scenario.
